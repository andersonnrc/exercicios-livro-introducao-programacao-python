## 📖 Sobre o projeto

Este repositório contém exercícios desenvolvidos com base no livro:

> **Introdução à Programação com Python – 4ª Edição: Algoritmos e lógica de programação para iniciantes**

O objetivo deste projeto é servir como material de apoio para estudo, prática e consolidação dos conceitos fundamentais de programação utilizando Python.

---

## 🎯 Objetivos

- Praticar lógica de programação
- Aprender os fundamentos da linguagem Python
- Resolver exercícios propostos no livro
- Criar uma base sólida para estudos mais avançados

---

## 🧠 Conteúdos abordados

- Variáveis e tipos de dados
- Estruturas condicionais (`if`, `else`, `elif`)
- Estruturas de repetição (`for`, `while`)
- Listas, tuplas e dicionários
- Funções
- Entrada e saída de dados
- Manipulação de strings
- Boas práticas de programação

---

## 🛠️ Ambiente recomendado

- Python
- Pylance

## 📌 Observações

Este repositório é voltado para aprendizado.
As soluções podem variar e não representam necessariamente a única forma correta.
Melhorias e refatorações são bem-vindas ao longo do tempo.
